# Sudanese-Dialect Dataset for Detection of Inappropriate Content in Arabic Text

This dataset contains Sudanese Arabic text labeled for hate, abusive, and normal content. It was created as part of a study on automatic detection of inappropriate content in Arabic text. 

**Important:** This dataset focuses exclusively on the Sudanese dialect.

## Dataset Composition

The dataset is composed of two parts:

1. **Publicly available portion:**  
   - Data collected by the authors themselves.  
   - This portion is **fully available** in this repository.  

2. **Portion obtained from other researchers:**  
   - Data obtained upon request from other researchers.  
   - Access to this portion requires contacting the original authors.  

### File Format

- **Format:** CSV  
- **Columns:**
  - `commentaire`: the text content
  - `classe`: label for the content
    - `0` = normal
    - `1` = abusive
    - `2` = hate  

**Notes:** Currently, there are no separate training/testing splits; all data is provided in the files.

---

## Usage

This dataset can be used for research on natural language processing, text classification, hate speech detection, and abusive language detection in Sudanese Arabic. Users should cite this dataset if used in any publications.
